#帮助系统

帮助系统的帮助    C-h ?

Emacs 快速指南    C-h t

Emacs使用手册    C-h r

Emacs说明    C-h i

搜索命令    C-h a

查看变量说明    C-h v

查看函数说明    C-h f

查看当前mode的相关文档    C-h m

"M-x describe-variable", 快捷键"C-h v", 查看变量的文档

"M-x describe-function", 快捷键"C-h f", 查看命令的文档

"M-x describe-key", 快捷键 "C-h k", 查看快捷键的文档



#标签

设置标签    C-x r m    

跳转标签    C-x r b    





#窗口管理

水平分割    C-x 2

竖直分割    C-x 3

只保留当前窗格    C-x 1

关闭当前窗格    C-x 0

切换窗格    C-x o

滚动下一个窗格    C-M-v

在另外一个窗口打开文件    C-x 4  b  



#缓冲区管理

查看缓冲区列表C-x C-b

切换缓冲区C-x b 

在两个缓冲区之间切换 C-x b RET

关闭缓冲区C-x k

保存多个缓冲区C-x s  

可成批删除缓冲区M-x kill-some-buffers

设置缓冲区为只读    C-x C-q  



#移动

向前移动一屏C-v    

向后移动一屏M-v    

向右移动一个字符C-f    

向左移动一个字符C-b    

向右移动一个词M-f    

向左移动一个词M-b    

移动到下一行C-n    

移动到上一行C-p    

移动到行首C-a    

移动到行尾C-e    

开头M-<    

末尾M->    

移动到指定行n    M-g M-g [n]    



#删除

删除光标后的一个字符C-d          

删除光标后的一个词M-d          

删除从光标到“行尾”间的字符C-k          



#搜索

搜索 C-s        

按C - s会跳到下一个结果

按C - r会跳到上一个结果

替换    M-x replace-string



#其他操作

另存为文件C-x C-w

退出 C-x C-c

终止命令    C-g

显示行号    M-x linum-mode   

与前一个字符交换    C-t     

寻找一个文件    C-x C-f  

储存这个文件    C-x C-s  

撤销    C-/

注释    M-;
